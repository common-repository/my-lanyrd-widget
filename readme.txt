== Description ==

My Lanyrd imports your Lanyrd.com profile into your WordPress blog. The Widget is based on source code originally developed by Lachlan Hardy (http://github.com/lachlanhardy/lanyrd-badge) using jQuery, which screen scrapes the HTML from Lanyrd using YQL.

The functioning code is Copyright (c) 2010 Lachlan Hardy. The full License can be found at: http://github.com/lachlanhardy/lanyrd-badge/blob/master/LICENSE which is the Expat License (aka MIT License). This is a simple, permissive non-copyleft free software license, compatible with the GNU GPL.

== Installation ==

Upload the My Lanyrad Widget to the plugin directory of your blog, Activate it, then add it to your sidebar. Be sure to enter your twitter account name and save.

1, 2, 3: You're done!
